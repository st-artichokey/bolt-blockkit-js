## Block Kit Feature Implementation

### Build retrospective modal with Block Kit inputs

Created a global shortcut handler that opens a modal demonstrating 6 Block Kit input types:
- `plain_text_input` — retro title and multiline feedback fields
- `static_select` — sprint/iteration picker
- `datepicker` — sprint end date
- `radio_buttons` — team mood selector
- `checkboxes` — focus area categories

### Handle modal submission and post summary

Built a submission handler that parses all form values and posts a rich Block Kit message to a configured channel. The summary uses header, section (with fields), divider, actions, and context blocks.

### Add chained modal and overflow actions

Implemented an "Add Comment" button that opens a secondary modal (demonstrating chained modals with `private_metadata` for thread context). Added overflow menu with pin-to-channel and bookmark options.

### Create App Home tab

Built an App Home view listing usage instructions and all Block Kit elements demonstrated in the app.

- **Sources:**
  - [Block Kit Interactive Components](https://api.slack.com/reference/block-kit/interactive-components)
  - [Slack Modals Guide](https://api.slack.com/surfaces/modals)
  - [Slack App Home Guide](https://api.slack.com/surfaces/app-home)
